# amplisim
Plain simple amplicon sequence simulator for in-silico genomic sequencing assays

## Installation
### Build from source
```
git clone --recursive https://github.com/Krannich479/amplisim.git
cd amplisim
mkdir build
make -C lib/htslib
make
```
